## コードレビューと提出

1. このリポジトリを **clone** する
1. `master` ブランチから新しいブランチを切り、作業を行う（例: `feat/lesson1`）
1. 課題を完了させたら、**Pull Request** を作成する
1. レビュー後にフィードバックがあれば、修正して再度 push する
1. Pull Request のコメントは自由で、気軽に難しかった点などを書いてください
